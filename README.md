# Лучшие школы Москвы — карта

Визуализация рейтинга лучших школ Москвы 2025 на OpenStreetMap.
Данные берутся со страницы [krylatskoye.ru](https://www.krylatskoye.ru/content/ratings/2025/08/rejting-shkol-moskvy-2025.html).

## Что это делает

- Скрапит список ~400 топ-школ Москвы, разбитых на 6 групп по баллам
- Геокодирует каждую школу через Nominatim (только здания в Москве, без детских садов)
- Показывает здания на карте цветными маркерами по группам:
  - 🟢 Группа 1 (ТОП-20)
  - 🟩 Группа 2 (21–70)
  - 🟡 Группа 3 (71–170)
  - 🟠 Группа 4 (171–220)
  - 🔴 Группа 5 (221–300)
  - 🔵 Группа 6 (301–400)

## Требования

- Node.js 18+
- Ключи API не нужны

## Установка

```bash
npm install
```

## Запуск

### 1. Собрать данные

```bash
node scrape.js
```

Скрипт выполняется ~8–10 минут (400 школ × 1 сек задержка — ограничение Nominatim).
По завершении создаются файлы `schools.json` и `data.js`.

### 2. Открыть карту

```bash
npm run serve
```

Откроет браузер на `http://localhost:8080` автоматически.
Остановить сервер: **Ctrl+C** в терминале.

> Или в VS Code: запустите задачу **"Open viewer"** (меню Terminal → Run Task).

### Обновление данных

Просто перезапустите `node scrape.js` — файлы `data.js` и `schools.json` перезапишутся.

## Файлы

| Файл | Описание |
|------|----------|
| `scrape.js` | Скрапер + геокодер, запускается через Node.js |
| `index.html` | Браузерная карта |
| `schools.json` | Данные о школах (генерируется) |
| `data.js` | Данные для браузера (генерируется) |
